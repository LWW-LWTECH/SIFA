Visibility · Validation

Conditional Fields & Live Validation

Selecting an account type reveals relevant sub-fields. Each field validates in real time, and the outcome panel reflects SIFA.outcome on every change.

  • showElement / hideElement on CHANGE
  • answerEquals condition check
  • Validation rules with true / false messages

condition: sifa_triggeredField("business")
true_actions: [
    sifa_hideElement("group_account_companyname"),
    sifa_hideElement("group_account_taxid"),
    sifa_hideElement("group_account_organisationname"),
    sifa_hideElement("group_account_charitynumber"),
    sifa_showElement("group_account_companyname"),
    sifa_showElement("group_account_taxid")
]
